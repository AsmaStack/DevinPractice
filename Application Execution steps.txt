## 🚀 Quick Start (3 steps)

### **Step 1: Install Dependencies**
```bash
cd c:\Users\HP\OneDrive\TaskFlow
npm install
```

### **Step 2: Start Backend (json-server)**
```bash
npm run start-server
```
Leave this terminal running. You should see:
```
Resources
http://localhost:3000/tasks
```

### **Step 3: Open the App**
Open index.html in your browser, or:
```bash
start public\index.html
```

## 🐛 The 3 Intentional Bugs

| Bug | Location | Observation | Difficulty |
|-----|----------|-------------|-----------|
| **#1: Duplicate API Calls** | app.js, lines 75-89 | Open DevTools Network tab on page load - see 2 GET requests | ⭐ Easy |
| **#2: Duplicate Entries on Add** | app.js, addTask() | Add a task - appears twice until refresh | ⭐⭐ Medium |
| **#3: UI State Mismatch** | app.js, toggleTaskCompletion() | Use DevTools throttling, toggle task, UI updates before server | ⭐⭐⭐ Hard |
